Software Engineering Basics

The core concepts stripped down to their simplest terms.

Software Engineering is the process of building, testing, and maintaining computer programs using a structured, step-by-step plan.

The 6 Core Aspects

  • RequirementsThe "What"

    Figuring out exactly what problem needs to be solved and what the software must do.

  • DesignThe "Blueprint"

    Planning how the software will look and how the different parts will connect before actually building it.

  • DevelopmentThe "Building"

    Writing the actual code and instructions that make the software work.

  • TestingThe "Checking"

    Running the program to find and fix mistakes (bugs) to make sure it works correctly.

  • DeploymentThe "Shipping"

    Sending the finished software out into the world so real people can actually use it.

  • MaintenanceThe "Upkeep"

    Updating the software over time to keep it secure, fix new problems, and add new features.
